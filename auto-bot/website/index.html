How it works

Three steps from inventory
to live listing

No training, no setup. If you can open Chrome, you're ready.

1

Install the extension

Add PostBot to Chrome from the Web Store — one click, no configuration required. Works on any Windows or Mac computer.

2

Open any vehicle page

Navigate to a vehicle on your dealership website and click the PostBot icon. The AI reads the page and extracts all listing details and photos automatically.

3

Review and post

PostBot pre-fills the Facebook Marketplace form: price, year, make, model, mileage, description, and images. Review, make any edits, and hit post.
